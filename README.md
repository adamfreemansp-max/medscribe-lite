# medscribe-lite

MedScribe Lite — Clinical transcription assistant using Whisper speech-to-text with medical terminology optimisation.

## Part of Voltron v5

This project is one of 5 healthcare software streams managed by the Voltron automation system.

| Stream | Description |
|--------|-------------|
| gp-medical-suite | GP clinical workflow + Truth Shield |
| jrny-health | Patient health journey tracker |
| cpd-companion | Continuing professional development |
| medinfo-hub | Medicine information platform |
| medscribe-lite | Clinical transcription assistant |

## Tech Stack

- **Frontend**: React / Next.js
- **Backend**: Node.js / Python
- **AI**: Claude API + Ollama (local models on Gigabyte AI TOP)
- **Infrastructure**: Mac Studio (brain) + Gigabyte 128GB VRAM (muscle)

## Getting Started

```bash
npm install
npm run dev
```
